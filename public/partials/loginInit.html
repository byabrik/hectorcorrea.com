Initialize Admin User

Use this button to initialize the first user in the database.

The new user will be created with the values defined in the environment variables defined on the server for BLOG_USER, BLOG_PASSWORD, and BLOG_SALT.

Typically this means that you ran the server as follow:

  $ BLOG_USER=uu BLOG_PASSWORD=pp BLOG_SALT=ss node server.js

You can only initialize the admin user if there are no other users in the database.
